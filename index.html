NiFi Helm Chart

소개

Apache NiFi를 Kubernetes 클러스터에 배포하기 위한 Helm Chart입니다. 이 Helm Chart를 사용하면 NiFi를 손쉽게 설치하고 관리할 수 있습니다.

특징

  • 간편한 설치 및 업그레이드
  • 유연한 구성 옵션
  • 확장 가능한 아키텍처

설치 방법

  1. Helm 저장소 추가: 
    helm repo add kurly https://helloworld.kurly.com/helm-nifi
  2. Helm 차트 설치: 
    helm install nifi kurly/nifi
